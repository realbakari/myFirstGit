# myFirstGit

Week 2 Workshop

## Installation

if you really want to edit this project, you need clone this project and make sure you have install Node dependencies. To do so, go to the project directory (from your computer root directory):

```bash
$ cd myFirstGit
```

and then run:
```bash
$ node index.js
```
